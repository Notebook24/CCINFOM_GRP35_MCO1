0. Change the credentials in the DBConnection.java to your credentials of your MySQL account.

1. To compile the program in the Command Line, Use:
javac AppDriver.java

2. To run the program in the Command Line, Use:
java -cp ".;mysql-connector-j-9.5.0.jar" AppDriver

3. If you are using IDEs such as Visual Studio Code, do not simply click the Run button.
- Go to AppDriver.java
- Click the Arrow down beside the run button
- Choose "Run Java" so that the app is actually connected to the mysql-connector jar file.
